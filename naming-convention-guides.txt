Controller => Pascal Case 
    - Example : SampleController

Class Methods => Camel Case
    - Example : public function addFunctionality();

Variables => Camel Case
    - Example : $sampleVariable

Class => Pascal Case
    - Example : PrintPersonalDataSheetController

Routes => Lower case
    - Example : Route::get('sample/controller', 'PrintPersonalDataSheetController');

Migration & Tables => Snake Case
    - Example : create_sample_migrations
